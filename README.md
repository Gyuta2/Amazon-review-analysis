# Amazon-review-analysis
아마존 리뷰 데이터를 베이스로 감성분석을 통하여 최종적으로 긍정, 부정을 판단하는 프로젝트입니다.
정확도 향상을 위해서 각 모델별로 정확도를 확인하여 가장 높은 정확도의 모델을 이용할 예정입니다.
